The Problem: LLM Context Pollution

Five failure modes that make long AI conversations unreliable ‚Äî and they're all backed by peer-reviewed research.

ü™û

Sycophancy

RLHF-trained models tell you what you want to hear. Sycophancy increases as you optimize for human approval ‚Äî the model literally lies to please you.

üé≤

Prompt Sensitivity

Up to 76 accuracy points of difference from formatting changes alone. Caps, emotional language, and whitespace all shift output quality dramatically.

üóëÔ∏è

Context Contamination

Every message persists with equal weight ‚Äî including wrong turns, corrections, contradictions, and emotional outbursts. There's no "strike from the record."

üï≥Ô∏è

Lost in the Middle

Models attend best to the start and end of context. Critical decisions in messages 15‚Äì35 of a 50-message chat sit in the worst possible position.

Compounding Errors

At 99% per-decision accuracy, 200 sequential decisions yield only a 13.4% chance of zero errors. Context pollution drops that accuracy far below 99%.

Part 2

Mike's Way: Clean Context Management

The core insight: the conversation is not the context. Conversations are how humans think ‚Äî messy, iterative, full of wrong turns. That's fine for thinking. It's terrible for instructing an AI.

‚úÇÔ∏è

Separate Conversation from Context

Have the messy conversation. Extract only the final decisions. Write them clean. Feed the clean version forward. Never pass raw chat to production prompts.

üìù

Rewrite Everything

After decisions are made, rewrite output into structured documents that strip all noise, corrections, and emotional artifacts.

üéØ

Model-Aware Writing

Write context to match the consuming model. Opus gets reasoning prompts. Codex gets concrete specs. Qwen gets explicit constraints.

üìê

Small, Focused Context

Target the 2K‚Äì8K token sweet spot. A focused PRD beats a 50K conversation dump every time.

üìÅ

Rules as Files

Keep behavioral rules in standalone files injected fresh each session ‚Äî never contaminated by conversational drift.

üî•

Pipeline Firebreaks

Each pipeline stage gets clean context. Model transitions reset sycophancy, strip emotions, resolve contradictions.
